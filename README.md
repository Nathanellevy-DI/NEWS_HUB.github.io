# Everything API Hub

A comprehensive website integrating multiple APIs to display useful information with custom styling for each feature.

## Project Structure

```
everything-api-hub/
├── index.html          # Main HTML file
├── css/
│   ├── main.css       # Global styles
│   └── features/      # Feature-specific styles
├── js/
│   ├── main.js        # Main application logic
│   └── apis/          # API integration modules
└── assets/            # Images, icons, etc.
```

## APIs Integrated

APIs will be added as they are provided.

## Getting Started

Simply open `index.html` in a web browser to view the application.

## Development

This project uses vanilla HTML, CSS, and JavaScript for maximum flexibility and performance.
